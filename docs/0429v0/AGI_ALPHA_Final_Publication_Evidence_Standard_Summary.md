# Evidence standard summary

A valid Evidence Docket contains task manifests, baselines, AGI ALPHA run traces, proof bundles, replay logs, cost ledgers, safety ledgers, validator reports, alpha-WU calibration, RSI state hashes, ECI ledger, Move-37 dossiers where relevant, summary tables, and independent replay notes. Internal demos, repository activity, and local/devnet proof surfaces count as implementation evidence; empirical performance requires comparative, replayable, real-task evidence.
