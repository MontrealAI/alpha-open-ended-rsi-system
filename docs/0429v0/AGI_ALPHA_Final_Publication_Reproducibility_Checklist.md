# Reproducibility checklist

1. Task manifest published or auditable.
2. Baselines declared before evaluation.
3. Agent constellation and role contracts logged.
4. Tool calls scoped and traceable.
5. ProofBundle complete.
6. Validator reports attributable.
7. Safety ledger includes blocked and quarantined actions.
8. Cost ledger includes tokens, compute, tools, latency, and human review.
9. RSI state hashes preserved.
10. Replay instructions provided.
11. Failures reported.
12. Promotion rule applied mechanically.
