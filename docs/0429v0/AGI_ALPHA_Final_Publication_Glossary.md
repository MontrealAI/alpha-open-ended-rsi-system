# Glossary

**AGI ALPHA**: commercially independent substrate for intelligence organizations.  
**alpha-AGI Ascension**: proposed operating regime for verified, risk-bounded agentic work.  
**ProofBundle**: settlement-grade proof object.  
**alpha-Work Unit / alpha-WU**: policy-parameterized unit of verified machine labor.  
**MandateEpoch**: epoch-level unit for batched, validated invention work.  
**NettingHouse**: batching and clearing layer for receipts and archive deltas.  
**Paymaster**: funding rail for mandate epochs.  
**OpenClaw**: human-facing sponsor/reviewer/operator shell.  
**Alpha-Factory**: invention engine for proposal, causalization, scoring, archive, and promotion.  
**AGI Node**: verifiable runtime for generation, evaluation, replay, and validation.  
**AGI.Eth / ASI.Eth**: low-entropy institutional namespaces.  
**RSIState**: deterministic state object for recursive invention governance.  
**ECI**: Evidence Contact Index.  
**Move-37 dossier**: high-novelty breakthrough package requiring reproduction and stress tests.  
**Evidence Docket**: public or internal empirical proof package.  
**Owner-operator flywheel**: verified work to capability to capital/infrastructure to stronger work.
