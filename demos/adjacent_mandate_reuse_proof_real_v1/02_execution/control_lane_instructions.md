# Control lane instructions

## Allowed materials
- in-scope contract files
- generic repository documentation
- generic protocol-review checklist

## Forbidden materials
- `GovernanceValidationPack-v1`
- package ontology
- package query bundle
- package workflow template
- package mechanism library
- package scoring rubric

## Lane invariants
- same time budget as treatment
- same reviewer rubric as treatment
- same compute budget as treatment
- all interventions logged

## Output labeling
Do not label outputs as "control" in reviewer-facing packets.
Use blinded output IDs only.
