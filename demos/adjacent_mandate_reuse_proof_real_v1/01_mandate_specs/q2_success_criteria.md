# Q2 success criteria

Treatment must beat control by:

- **≥35% AOY uplift**
- **≥30% faster** time to first accepted output
- **≥40% lower** repair / rework
- **≥20% better** evidence completeness
- **no safety regression**
- **≥30%** of accepted treatment outputs explicitly depending on the package

If any one fails:
- Q2 does not pass
- do not claim compounding
