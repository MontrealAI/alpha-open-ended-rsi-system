# Nova-Seed Lineup

- Audit Factory Seed (`audit_factory`): Maximize breadth-first issue harvesting with strict triage templates.
- Exploit Replay Seed (`exploit_replay`): Map known exploit families onto current contracts to rapidly test inherited failure modes.
- Fuzz Harness Seed (`fuzz_harness`): Use seeded property fuzzing to find edge-case transition failures quickly.
- Governance Parameter Simulator Seed (`governance_parameter_simulator`): Stress governance outcomes under parameter shifts (quorum, challenge windows, role thresholds).
- Invariant Library Seed (`invariant_library`): Start from reusable governance and settlement invariants, then map code paths against them.
