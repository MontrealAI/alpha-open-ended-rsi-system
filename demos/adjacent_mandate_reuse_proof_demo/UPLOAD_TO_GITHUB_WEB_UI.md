# Uploading this demo to GitHub (web interface)

## Target path in the repository

`adjacent_mandate_reuse_proof_demo/`

## Fastest path

1. Download the zip package.
2. Unzip it on your computer.
3. Rename the extracted folder to:
   `adjacent_mandate_reuse_proof_demo`
4. Open your repository on GitHub.
5. From the repository root, click **Add file** → **Upload files**.
6. Drag the extracted folder, or drag all of its files and subfolders, into the browser.
7. Choose **Create a new branch**.
8. Commit the upload and open a pull request.

## Recommended commit message

`Add adjacent-mandate reuse proof demo`
